优化编译 Xkeyopt

作者： GaGe / 2022-12-04 / 分类： knowledge

knowledge

翻译： EN

**-Xkeyopt=rtopt / -Xkeyopt=tbopt

可以增强vcs 编译优化

原文：

Many runtime performance improvements or optimizations in this release are enabled using the -Xkeyopt=rtopt option. Also, many runtime performance improvements for UVM testbenches are enabled using the -Xkeyopt=tbopt option. The optimization applies to UVM factory, resource pool and regular expression operations.
